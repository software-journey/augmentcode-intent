# Example tracks

Wave 2 turned the example tracks into guided practice workbooks. Wave 3 added small starter implementations, Wave 4 extended each with a realistic follow-up slice, and Wave 5 matures both into production-style local applications.

Choose one track and follow it through Episodes 2-8, or compare both tracks to see how the same workflow applies in different domains.

| Track | Best for | Start here |
| --- | --- | --- |
| `jwt-workshop/` | Backend/auth-oriented practice | `jwt-workshop/README.md` |
| `delivery-dashboard/` | Product/UI-oriented practice | `delivery-dashboard/README.md` |

Each track now includes worked artifacts for:

- living specs
- coordinator/specialist handoffs
- isolated workspace practice
- spec seeds
- context exercises
- orchestration waves
- PR-ready summaries

Each track also now includes a runnable implementation path:

- `jwt-workshop/starter-api/` for the Express + TypeScript auth + shipment API with SQLite persistence
- `delivery-dashboard/starter-app/` for the React + TypeScript dashboard with routed auth and real API-backed shipment data
